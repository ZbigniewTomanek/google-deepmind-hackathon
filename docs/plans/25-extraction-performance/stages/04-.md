# Stage 4: Pre-Computed Entity Embeddings for Librarian

**Goal**: Batch-embed all extracted entity descriptions before the librarian runs, so `create_or_update_node` can skip individual `embed()` calls for matching descriptions.
**Dependencies**: None (independent of Stages 1-3)

---

## Context

Currently, the librarian's `create_or_update_node` tool (agents.py:714-715) calls `embed(content)` for every entity it creates/updates. With 10+ entities per episode, this adds ~5-10s of sequential embedding API calls.

The extractor provides entity descriptions that the librarian often passes through unchanged. By pre-computing embeddings for all extracted descriptions in a single `embed_batch()` call, we can serve most of these from a cache dict.

When the librarian modifies a description, the cache misses and `embed()` is called as before — correctness is preserved.

---

## Steps

1. **Add `precomputed_embeddings` field to LibrarianAgentDeps**
   - File: `src/neocortex/extraction/agents.py`
   - Find the `LibrarianAgentDeps` dataclass and add:
     ```python
     precomputed_embeddings: dict[str, list[float]] = field(default_factory=dict)
     ```
   - This maps `content_text -> embedding_vector`.

2. **Pre-compute embeddings in pipeline.py before librarian call**
   - File: `src/neocortex/extraction/pipeline.py`
   - Find the tool-driven librarian branch (search for `if librarian_use_tools:`). Between the extractor result and the `LibrarianAgentDeps(` constructor call, add:
     ```python
     # Pre-compute embeddings for extracted entity descriptions (single batch call)
     precomputed_embeddings: dict[str, list[float]] = {}
     if embeddings:
         descriptions = [e.description for e in extraction_result.output.entities if e.description]
         if descriptions:
             batch_results = await embeddings.embed_batch(descriptions)
             for desc, emb in zip(descriptions, batch_results, strict=True):
                 if emb is not None:
                     precomputed_embeddings[desc] = emb
     ```
   - Pass `precomputed_embeddings=precomputed_embeddings` to the `LibrarianAgentDeps(...)` constructor in the tool-driven branch.

3. **Use cache in create_or_update_node tool**
   - File: `src/neocortex/extraction/agents.py`, lines 713-715
   - Change:
     ```python
     embedding = None
     if ctx.deps.embeddings and content:
         embedding = await ctx.deps.embeddings.embed(content)
     ```
   - To:
     ```python
     embedding = None
     if ctx.deps.embeddings and content:
         embedding = ctx.deps.precomputed_embeddings.get(content)
         if embedding is None:
             embedding = await ctx.deps.embeddings.embed(content)
     ```
   - This is a simple dict lookup — O(1), no hashing overhead beyond Python's built-in str hash.

4. **Handle the non-tool (fallback) librarian path**
   - File: `src/neocortex/extraction/pipeline.py`
   - Find the `else` branch (fallback non-tool librarian path). This path already uses `embed_batch()`, so no embedding changes needed there.
   - But `LibrarianAgentDeps` is also constructed in this branch — pass an empty dict for `precomputed_embeddings` to satisfy the field, or rely on the `field(default_factory=dict)` default.
