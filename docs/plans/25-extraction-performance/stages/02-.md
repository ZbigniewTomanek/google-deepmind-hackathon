# Stage 2: Parallel Metadata Fetches

**Goal**: Replace 3 sequential DB calls at pipeline start with a single `asyncio.gather`.
**Dependencies**: None (independent of Stage 1)

---

## Steps

1. **Parallelize get_node_types, get_edge_types, get_type_examples**
   - File: `src/neocortex/extraction/pipeline.py`, lines 91-99
   - Current code (sequential):
     ```python
     node_types = await repo.get_node_types(agent_id, target_schema=target_schema)
     edge_types = await repo.get_edge_types(agent_id, target_schema=target_schema)
     # ... build dicts ...
     type_examples = await repo.get_type_examples(agent_id, target_schema=target_schema)
     ```
   - Replace with:
     ```python
     import asyncio
     node_types, edge_types, type_examples = await asyncio.gather(
         repo.get_node_types(agent_id, target_schema=target_schema),
         repo.get_edge_types(agent_id, target_schema=target_schema),
         repo.get_type_examples(agent_id, target_schema=target_schema),
     )
     ```
   - Then build the description dicts after the gather (lines 95-96 stay the same, just move after).

2. **Add `import asyncio`** at top of `pipeline.py` if not already present.
