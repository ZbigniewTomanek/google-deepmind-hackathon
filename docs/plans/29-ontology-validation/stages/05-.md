2. **Wait for all extraction jobs to complete**

3. **Check total episode count** — we should have 13 episodes total (8 from Stages 2-4 + 5 from this stage).
   The consolidation trigger runs every 10 episodes, so it should have fired at least once.
   ```bash
   # Check job count
   curl -s http://localhost:8001/admin/jobs/summary \
     -H "Authorization: Bearer admin-token" | python3 -m json.tool
   ```

4. **Check consolidation logs** — verify the merge map and archive ran
   ```bash
   # Look for consolidation activity
   grep -i "consolidat" log/agent_actions.log
   grep -i "merge" log/agent_actions.log
   grep -i "archive" log/agent_actions.log
   ```

5. **Verify merge map application** — check if any of the 8 merge mappings were exercised
   ```bash
   # Check if source types from the merge map still exist (they shouldn't have nodes)
   docker compose exec postgres psql -U neocortex -d neocortex -c \
     "SELECT schema_name FROM graph_registry ORDER BY schema_name;"
   ```

   For each schema, check:
   ```bash
   docker compose exec postgres psql -U neocortex -d neocortex -c \
     "SELECT name FROM {schema}.node_type
      WHERE name IN ('AnatomicalLocation', 'AnatomicalStructure', 'HealthActivity',
                     'SportActivity', 'ArchitectureConcept', 'TechnicalArchitecture',
                     'HealthState', 'Condition')
      ORDER BY name;"
   ```

6. **Check unused type archiving** — verify types with 0 usage (excluding seeds) were cleaned up
   ```bash
   # For each schema, count unused non-seed edge types
   docker compose exec postgres psql -U neocortex -d neocortex -c \
     "SELECT et.name
      FROM {schema}.edge_type et
      LEFT JOIN {schema}.edge e ON e.type_id = et.id
      WHERE e.id IS NULL
      ORDER BY et.name;"
   ```

7. **Run full metrics** — use all-in-one query from `resources/queries.md` for each schema

8. **Check type reuse** — with 13 episodes, types should be getting reused
   ```bash
   docker compose exec postgres psql -U neocortex -d neocortex -c \
     "SELECT
        COUNT(n.id) AS total_nodes,
        COUNT(DISTINCT n.type_id) AS active_types,
        ROUND(COUNT(n.id)::numeric / NULLIF(COUNT(DISTINCT n.type_id), 0), 1) AS reuse_ratio
      FROM {schema}.node n;"
   ```
