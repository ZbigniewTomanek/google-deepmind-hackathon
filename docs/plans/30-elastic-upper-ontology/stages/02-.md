# Stage 2: Baseline Experiment

**Goal**: Run the full 12-document corpus on a fresh installation and capture baseline metrics for all success criteria.
**Dependencies**: Stage 1 DONE

---

## Steps

1. **Fresh start**
   - `./scripts/manage.sh stop --all && ./scripts/manage.sh start --fresh`
   - Confirm 4 seed domains via Q1

2. **Ingest all 12 documents sequentially (no delay)**
   - Run all 12 curl commands from `resources/commands.md` back-to-back
   - No delay between requests — let the job queue process naturally

3. **Wait for all jobs to complete**
   - Poll via: `psql ... -c "SELECT status, count(*) FROM public.jobs GROUP BY status;"` every 10s
   - Wait until no `pending` or `running` jobs remain (timeout after 5 minutes)
   - Check for `failed` jobs — log any failures

4. **Capture baseline metrics**

   Run each diagnostic query from `resources/queries.md` and record results:

   **Q1: Domain registry**
   - Expected: exactly 4 seed domains, 0 new domains
   - Record: total domain count, any non-seed domains

   **Q2/Q3: Episode and node counts per domain**
   - Record: episode count, node count, edge count per domain schema
   - Note which domains got zero episodes

   **Q4: Catch-all absorption rate**
   - Record: % of episodes in `domain_knowledge` vs total
   - Expected: high % — novel content all goes to catch-all

   **Q5: Type landscape per domain**
   - Record: active node types, active edge types per domain
   - Note any new types created beyond seeds

5. **Analyze novel-domain behavior**

   For each Tier 3 document (Docs 07–12), determine:
   - Which domain(s) it was routed to
   - Whether any new domain was proposed (check logs)
   - Whether `domain_knowledge` absorbed it

   Check logs for domain classification decisions:
   ```bash
   grep -i "domain_classification\|proposed_domain\|route_episode" log/ingestion.log | tail -40
   ```

6. **Record baseline in this file**

   Fill in the baseline capture template from `resources/queries.md` and paste
   results below in the Results section.
